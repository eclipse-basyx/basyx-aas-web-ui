# BaSyx AAS Web UI Roadmap

The following features are planned for the AAS Web UI over the next quarters.
Please note that this list is not exhaustive and may be subject to re-prioritisation.

## Planned Features and Extensions

- Management interface for access rules (AAS Spec Part 4) (Q3/2026)
- UI components as dedicated graphical clients for all BaSyx components (Repositories, Registries, AAS Discovery) (Q4/2026)

## Improvements

- Better type support for the AAS (AAS Core Works) (Q2/2026)
- Usage of the BaSyx TYpeScript SDK for API interactions (Q2/2026)
- Improved quality management through testing via Vitest (Q3/2025)
